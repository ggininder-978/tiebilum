# Superpowers Skills Index: 技能索引表

以下是 Superpowers 內建的核心技能清單，作為 `Tiebilum OS` 提升自動化能力的參考。

| 技能名稱 | 核心功能 | 適用場景 |
| :--- | :--- | :--- |
| `using-superpowers` | **引導與分發** | 對話開始時，建立調用技能的意識。 |
| `brainstorming` | **需求釐清** | 開始任何新功能、組件或行為修改前。 |
| `writing-plans` | **任務拆解** | 設計批准後，實作開始前。 |
| `test-driven-development` | **質量保障** | 具體編碼階段，強制「測試先行」。 |
| `systematic-debugging` | **根因分析** | 修復 Bug 時，遵循四階段排查流程。 |
| `subagent-driven-development` | **並行開發** | 需要快速迭代多個任務時。 |
| `executing-plans` | **批次執行** | 按照計畫順序穩定推動開發。 |
| `requesting-code-review` | **內控審查** | 任務完成後，提交前的質量檢查。 |
| `verification-before-completion` | **完工驗證** | 宣稱任務完成前，最後的證據檢查。 |
| `writing-skills` | **技能開發** | 需要擴展 AI 能力（建立新技能）時。 |

## 技能分層
1.  **Process Skills (流程類)**：如 `brainstorming`, `debugging`。決定「如何切入任務」。
2.  **Implementation Skills (執行類)**：如 `frontend-design`, `mcp-builder`。引導具體編碼。
